# Jumbled-words-game
PROJECT TITLE: 

JUMBLED WORDS GAME

OVERVIEW OF PROJECT:

The Jumbled Words Game is a Python-based interactive game designed for two players. 
The game selects a random word from a predefined list, jumbles the letters, and challenges the players to guess the correct word. 
The player who correctly identifies the word scores a point. The game continues until the players choose to quit.

FEATURES:

Two-player interactive game.
Random word selection from a list of words
Shuffling/jumbling of selected word letters.
Turn-based gameplay.
Score calculation for each player.
Winner declaration at the end of the game.
Option to continue or quit after each round.

TECHNOLOGIES USED:

Programming Language: Python 3.x
Libraries Used: random (for selecting and shuffling words)
IDE/Environment: Any Python IDE or text editor (e.g. Google Colab)
