PROBLEM STATEMENT:
The Jumbled Words Game is a Python-based interactive game designed for two players. 
The game selects a random word from a predefined list, jumbles the letters, and challenges the players to guess the correct word. 
The player who correctly identifies the word scores a point. The game continues until the players choose to quit.

OBJECTIVE:
To create an engaging word game that improves vocabulary and cognitive skills.
To implement basic Python concepts like functions, loops, conditionals, and user input.
To practice handling turn-based logic and scoring mechanisms in a multiplayer game.

SCOPE OF THE PROJECT:
Understanding of random module for generating random choices.
Practical experience with functions and loops in Python.
Implementation of turn-based logic and score tracking.
Improved problem-solving skills through word jumbling logic.

TARGET USERS:
All the users who are above the age of 15 interested in solving puzzles and challenges can play the game

FEATURES:
Two-player interactive game.
Random word selection from a list of words.
Shuffling/jumbling of selected word letters.
Turn-based gameplay.
Score calculation for each player.
Winner declaration at the end of the game.
Option to continue or quit after each round.
